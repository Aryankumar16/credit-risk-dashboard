# How to Run

## Step 1 — Install Dependencies
pip install -r requirements.txt

## Step 2 — Make Sure the Datasets Are in the Same Folder
Ensure both files are present in the project directory:
- credit_risk_dataset.csv
- german_credit_data.csv

## Step 3 — Open the Notebook
jupyter notebook credit_risk_dashboard.ipynb

## Step 4 — Run All Cells
Go to Kernel → Restart & Run All

## Step 5 — Interact with the Dashboard
Once the final cell runs, the dashboard will open automatically
in your browser. Use the sidebar filters to explore the data.

## Requirements
- Python 3.8 or above
- Jupyter Notebook or JupyterLab
